# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Scope

This Code of Conduct applies in all project spaces — GitHub issues, pull requests,
discussions, the project's communication channels, and any public representation of
the project. It applies to all contributors, maintainers, and participants.

## Enforcement

**Contact:** conduct@accelmars.com

Reports are reviewed within 48 hours. All reports are treated as confidential.
The enforcement team will review each report and determine an appropriate response.

Project maintainers follow this enforcement ladder:

| Level | Response |
|-------|----------|
| 1 — Correction | Private written notice with explanation of the issue |
| 2 — Warning | Formal warning with specified behavior requirements and consequences |
| 3 — Temporary ban | Temporary suspension from project spaces for a defined period |
| 4 — Permanent ban | Permanent removal from all project community spaces |

Decisions by the enforcement team are final.
